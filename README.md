# GymBag - Trainer Management System

A comprehensive web-based management system designed for personal trainers and fitness professionals to manage clients, sessions, packages, and revenue tracking.

## Features

### 📅 Session Management
- Schedule and manage training sessions
- Create recurring session series
- Track session completion and status
- Calendar view for easy scheduling

### 👥 Client Management
- Complete client profiles with contact information
- Package tracking and session credits
- Client performance metrics
- Recent session history

### 💰 Revenue Tracking
- Financial ledger with detailed transaction history
- Package payment tracking
- Overdue payment alerts
- Revenue analytics and trends

### 📊 Business Analytics
- Dashboard with key performance metrics
- Client retention statistics
- Revenue trend analysis
- Session completion rates

### 🎯 Prospect Pipeline
- Lead management system
- Follow-up tracking and reminders
- Conversion tracking from prospect to client

## Technical Details

- **Type**: Single-page HTML application
- **Framework**: Vanilla JavaScript with modern ES6+ features
- **Storage**: Local browser storage (localStorage)
- **Styling**: CSS3 with custom properties and responsive design
- **Mobile**: Optimized for mobile devices with touch-friendly interface

## Getting Started

1. Open `GymBag.html` in any modern web browser
2. The application will automatically initialize with sample data
3. Start adding your clients and scheduling sessions
4. All data is stored locally in your browser

## Key Functionality

- **Data Persistence**: All information is automatically saved to browser local storage
- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Real-time Updates**: Dashboard metrics update automatically as you add sessions and clients
- **Export Capabilities**: Financial data can be viewed and managed through the built-in ledger

## Browser Compatibility

- Chrome/Chromium (recommended)
- Firefox
- Safari
- Edge

## Data Management

The application uses browser localStorage for data persistence. Your data remains private and local to your device. To backup data, use your browser's export functionality or manually copy the localStorage data.

## Usage Tips

- Use the calendar view to quickly see your schedule
- Set up recurring sessions for regular clients
- Monitor the financial ledger for payment tracking
- Check the prospect pipeline regularly for follow-ups
- Use the mobile-optimized interface for on-the-go management
